### 速度/情绪修饰词

| 类别 | 词汇 |
|---|---|
| 速度 | Smooth（流畅）、Slow（慢速）、Fast（快速）、Gradual（渐进） |
| 情绪 | Cinematic（电影感）、Dreamy（梦幻）、Aggressive（激进）、Tense（紧张） |

### 运镜组合示例（最多2-3个，用+连接）

- `Orbit + Zoom In` — 环绕同时推进，展示产品/角色
- `Crane Up + Pan` — 升起同时横移，展示大场景
- `Dolly Zoom` — 希区柯克变焦，恐惧/震撼时刻
- `Low Angle + Slow Crane Up` — 低角度缓慢上升，史诗感
- `Aerial + Gradual Pull Out` — 航拍+拉远，展示规模

## 光影三层结构写法

**结构：**
```
光影：[①光源：主光类型和角度] + [②光行为：如何影响大气/材质] + [③色调：冷暖对比]
```

**示例（暴雨场景）：**
```
光影：暴雨逆光+建筑爆炸橙红+霓虹灯漫射（光源层），
雨雾柔化高光但强化阴影对比、丁达尔效应贯穿全片（光行为层），
冷蓝底调+霓虹紫红高光（色调层）。
```

**示例（仙侠场景）：**
```
光影：熔岩自发光+雷电闪弧（光源层），
热浪蒸腾形成半透明橙红光晕（光行为层），
金红暖色调+冷蓝剑光对比（色调层）。
```

## 风格/氛围修饰词

### 画面风格
- 电影级质感、胶片颗粒、浅景深
- 2.35:1宽银幕、24fps
- 黑白水墨风格 / 动漫风格 / 超写实
- 高饱和霓虹色调、冷暖对比
- 4K医学CGI、半透明可视化
- 超精细CG动画技术

### 情绪氛围
- 紧张悬疑 / 温暖治愈 / 史诗恢宏
- 喜剧风格，表情夸张
- 纪录片风格，旁白克制
- 暗黑奇幻 / 仙侠高燃
- 赛博朋克 / 末日废土

### 音频指导
- 背景音乐：恢宏大气 / 轻柔治愈 / 急促紧张
- 音效：走路声、人群声、汽车声
- 旁白音色参考 @视频1
- 转场画面与音乐节奏卡点
- 脚步声、呼吸声、衣料摩擦声清晰并与节拍贴合

## 收束句写法参考

收束句 = 后期处理词 + 张力宣言，为全片视觉风格做最终定格。

| 风格 | 收束句示例 |
|---|---|
| 赛博朋克 | 暗角+胶片颗粒+电子噪点混雨雾粒子收尾，窒息式压迫感，全程高张力 |
| 仙侠 | 音效收束为余音震颤与渐弱风声，镜头定格牌坊剪影 |
| 治愈 | 暖光渐入，音乐轻柔收束，画面在午后阳光中定格 |
| 商业广告 | 产品主视觉定格，品牌广告语出现，背景音乐推向高潮 |
| 科幻 | 镜头定格在星球表面反射的宇宙倒影，渐入黑屏 |
