## 一、美术风格总纲（Art Direction）

### 1.1 核心风格矩阵

| 风格标签 | 关键词 | 应用范围 |
|:---|:---|:---|
| **High-Fidelity 3D Anime** | 大眼、精致五官、动漫比例 | 人脸与角色 |
| **PBR写实质感** | 皮肤SSS、布料物理、金属反射 | 材质系统 |
| **最终幻想风格** | FF7R / FF16 人脸建模 | 角色面部参考 |
| **Cyberpunk + Wasteland** | 赛博朋克 × 废土 | 全场景 |
| **Weathered Rusty Robot** | 生锈、破旧、工业锈蚀 | 机甲/载具 |
| **Rusty Cybernetics** | 陈旧义体、磨损机械 | 机械臂/义肢 |
| **Mad Max Cyberpunk** | 疯狂麦克斯 + 霓虹赛博 | 载具、废墟 |
| **NieR Automata Wasteland** | 尼尔机械纪元废土 | 环境、机甲 |
| **Tactical Scavenger Outfit** | 战术拾荒者装扮 | 底层角色服装 |
| **Alien Desert Moons** | 多月亮异星沙漠 | 猎杀场远景 |
| **Blender + 3D Render** | 电影级3D渲染 | 技术管线 |

### 1.2 光影与色调规范

- **高维都市**：霓虹冷光 + 冷暖对比，赛博蓝紫
- **猎杀场**：血红色天空、毒圈绿光、锈蚀金属橙褐
- **黑市**：昏暗潮湿、荧光标牌、丁达尔体积光
- **系统空间**：流动0与1、纯黑背景、高亮数据流

### 1.3 负向排除（Negative Prompt 建议）

```
无CGI塑料感、无卡通扁平、无过度光滑金属、无干净崭新机甲、
无跳跃剪辑闪烁、无廉价特效、无低分辨率贴图
```
