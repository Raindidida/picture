## 二、BGM选曲参考方向

### 主选曲情绪节奏关键词
- **氛围感后摇**：缓慢堆叠 → 情绪爆发 → 坠落消散
- **Lo-Fi人声切片**：呢喃碎语 + 电话回响 + 磁带噪声
- **抖音热歌候选风格**：
  - 类《以父之名》结构感（前奏留白 → 中段迸发）
  - 类《光年之外》情绪弧度（遥远 → 靠近 → 割裂）
  - 类《漠河舞厅》时代感叙事
  - 类《花海》梦游质感

> **BGM节奏锚点划分（对应视频节奏）：**
> - 0:00–0:45 → 低沉留白期（建立世界）
> - 0:45–1:30 → 情绪酝酿期（人物入场）
> - 1:30–2:20 → 张力堆叠期（关系裂变）
> - 2:20–3:00 → 情绪破防点（核心高潮）
> - 3:00–3:30 → 消散离场期（余烬收尾）

## 三、MV故事梗概

> **标题：《你在的那年 空气是蓝色的》**

一个关于**十七岁夏天的意识残影**。

那个夏天没有名字，只有：
屋顶上的风，操场边的自行车，
一件没有还回去的外套，
以及某人转身离开时，
空气里弥漫的、无法命名的蓝色气息。

故事不讲究因果，只讲感觉的**质地**——
喜欢一个人是什么质地？
是透明的，是钝的，是刺入肺部的细针，
是你站在原地，但整个世界已经轻轻移位。

没有明确的男女主角关系定义。
只有**眼神、手、背影、光**，
以及所有你懂得的东西都不需要解释。

## 四、分镜大纲（共18段）

---

### 【第一章：世界的褪色质感】0:00–0:45

#### 镜头 01 — 开场定调
```
画面：极慢速推近一扇老旧木框窗户
窗玻璃起雾，手指在玻璃上划出一道痕迹
玻璃外是过曝的白色天空
色调：冷白底色 + 微蓝色散光
```
- 情绪：茫然 / 前情绪状态
- 叠文字（手写字体）：**"那一年的空气是蓝色的"**

---

#### 镜头 02 — 城市碎片
```
画面：超广角仰拍高楼，镜头倾斜15度
快速剪切：公交车窗外的虚焦街道 / 电线杆 / 晃动的树冠
画面轻微过曝，颗粒感明显
色调：橙黄暖调 + 绿色偏移
```
- 情绪：漂浮感 / 不属于任何地方
- 手持抖动，模拟记忆回溯质感

---

#### 镜头 03 — 学校走廊
```
画面：空旷走廊，顺光拍摄，逆光人影
一个背影走向光源端，越来越模糊
镜头缓慢变焦失焦，人影化为光斑
色调：青绿 + 黄白交接处
```
- 情绪：错过前的最后一刻
- 配文字：**"你走进光里，我就开始失明"**

---

### 【第二章：人物的浮现】0:45–1:30

#### 镜头 04 — 主角A 初现
```
画面：教室窗边侧脸，望向窗外
强烈侧逆光，脸部半明半暗
眼神失焦，睫毛上有细小的光点
色调：暖黄 → 蓝色过渡
特写：手指轻轻触摸课桌边缘
```
- 情绪：存在感极弱 / 自我消隐中的存在
- 叙述字幕：**"我总是很安静，不是因为没有话，是因为你不在"**

---

#### 镜头 05 — 主角B 出现
```
画面：操场边，逆光骑自行车
慢动作拍摄，车轮扬起的微尘在光线中漂浮
镜头过曝，人物边缘出现白色光晕
突然切：近景，B的耳朵上戴着耳机，眼睛微眯
色调：橙红 + 褪色白
```
- 情绪：无意识的存在感 / 让人窒息的普通
- 配文字：**"你只是骑着车经过，我就已经没法呼吸了"**

---

#### 镜头 06 — 图书馆相遇
```
画面：镜头透过书架缝隙拍摄
焦段极浅，书脊是前景虚焦色块
中间层：A抬头看向某处
远景：B站在书架另一侧，正在取书
二人隔着书架，视线方向不明确
色调：蓝灰 + 暖黄光斑
```
- 情绪：若即若离 / 观察的距离感
- 叙述字幕：**"所有的喜欢都发生在两毫米的空隙里"**

---

#### 镜头 07 — 雨中场景
```
画面：雨水打在窗玻璃上
镜头贴近玻璃拍摄，室外人影变成雨痕变形
一滴雨沿玻璃滑落，在它滑落的轨迹里，室外的树模糊变形
色调：蓝紫 + 灰白
```
- 情绪：时间的湿润感 / 说不出的接近
- 配文字：**"是雨水，是眼泪，我分不清楚"**

---

### 【第三章：情感的中层燃烧】1:30–2:20

#### 镜头 08 — 屋顶场景（核心场景）
```
画面：楼顶，黄昏前的蓝色天空
二人背对镜头坐在天台边缘（不直接展示脸）
B的头发被风吹乱
A的手在身侧，距离B的手只有五厘米
镜头缓慢下降，穿过栏杆，呈现虚焦的城市远景
色调：深蓝渐变 + 橙色地平线
```
- 情绪：最接近的瞬间 / 沉默的满溢
- 叙述字幕：**"我们沉默着，但那五厘米里住着整个宇宙"**

---

#### 镜头 09 — 外套
```
画面：A穿着B的外套，手插进口袋
极近特写：外套袖口比手腕长出一截
慢动作，风吹动衣角
色调：暖橙 → 冷蓝（表示穿上外套之前和之后）
```
- 情绪：代入感 / 气味与温度的想象
- 配文字：**"口袋里什么都没有，但我不想还给你"**

---

#### 镜头 10 — 耳机传递
```
画面：两个人坐着，B把一只耳机递给A
特写：手指接触的瞬间（慢动作）
手指接触时，画面轻微震动（特效：心跳感）
切：A戴上耳机，眼睛微微闭上
色调：全程低饱和 + 单点暖黄高光
```
- 情绪：共振 / 通过音乐进入对方的世界
- 叙述字幕：**"那首歌从此以后，是你的颜色"**

---

#### 镜头 11 — 走廊奔跑（快节奏切段）
```
画面：快速剪切 × 8帧
- 空走廊的残影
- 楼梯间逆光向上奔跑
- 操场边草地上倒下的自行车
- 黑板上被擦掉的字迹
- 手机屏幕亮起又熄灭
色调：颜色快速交替：青/橙/白/黑
```
- 情绪：青春的混乱能量 / 节奏爆发
- 此段对应BGM节奏最密集的切片部分

---

#### 镜头 12 — 镜子里的脸
```
画面：浴室镜子前
镜子里的A直视镜头（即直视自己）
画面轻微变形（广角畸变）
A的脸逐渐失焦，只剩轮廓
色调：冷白 + 荧光蓝
```
- 情绪：自我凝视 / 喜欢一个人时开始认不清自己
- 配文字：**"你的眼睛太亮了，亮得我照镜子也看不见自己"**

---

### 【第四章：撕裂与消散】2:20–3:00

#### 镜头 13 — 空位
```
画面：教室全景，B的座位是空的
慢慢推进，座位上光线打落，扬起轻微灰尘
特写：椅子表面的划痕
色调：全程极度去饱和 + 淡蓝冷调
```
- 情绪：缺席 / 已经开始的失去
- 叙述字幕：**"你不在的那天，太阳依然升起，我不能原谅太阳"**

---

#### 镜头 14 — 河边（离别前置场景）
```
画面：河边/湖边，傍晚
B站在岸边，背对镜头
慢动作，头发被风扬起
A在画面边缘，焦段外
镜头缓慢推远，B的身影越来越小
色调：橙金 → 蓝紫（暮色渐染）
```
- 情绪：不可避免的预感 / 最后的温度
- 配文字：**"你站在那里，我站在这里，中间是整条河流"**

---

#### 镜头 15 — 手机屏幕
```
画面：极近特写，手机屏幕
屏幕上是一段对话（内容模糊，只看到排版结构）
手指悬在屏幕上，没有按下发送
屏幕熄灭，黑屏映出持手机的手
色调：手机光 → 全黑
```
- 情绪：沉默的代价 / 所有没说出口的话
- 叙述字幕：**"我删了又写，写了又删，最后发出去的是：没事"**

---

#### 镜头 16 — 高潮情绪爆发
```
画面：回到屋顶，但这次是夜晚
A独自站在天台边，城市灯光在下方漂浮
仰拍：黑蓝夜空 + 城市光污染的橙色云层
慢动作，A转身，脸对着镜头，但眼神看穿镜头后方
风吹起衣角，出现轻微动态模糊
色调：深蓝 + 橙红光污染 + 高对比
```
- 情绪：整个MV情绪峰值
- 此处BGM进入主副歌高潮段落
- 配文字（全屏叠加，淡入淡出）：**"原来呼吸是蓝色的，我是在你走了以后才明白的"**

---

### 【第五章：余烬 · 消散 · 尚存】3:00–3:30

#### 镜头 17 — 意识流收尾
```
画面：多重曝光叠加
- 屋顶上的二人 × 空教室 × 雨水玻璃 × 奔跑的背影
所有画面在最后15秒叠在一起，缓慢淡出
色调：全程冷蓝，逐帧向白色过渡
```
- 情绪：记忆的总体坍缩 / 存在过的证明
- 叙述字幕叠入：**"那年夏天，我们都在，所以那年夏天，永远在"**

---

#### 镜头 18 — 结尾定格
```
画面：最终帧
回到镜头01：同一扇窗，玻璃上的划痕
但这次窗外不再是白色过曝的天空
而是一片纯粹深蓝色
画面完全静止3秒，缓慢淡黑
```
- 情绪：终局 / 蓝色即答案
- 最后字幕（白字黑底）：**"你在的那年 / 空气是蓝色的"**

## 五、视觉处理技术规格

| 元素 | 处理方式 |
|------|----------|
| 调色基调 | 阴影压蓝/青，高光推橙/黄，中间调去饱和 |
| 胶片效果 | 添加4K胶片颗粒 + 轻微色差（Chromatic Aberration） |
| 镜头质感 | 全程大光圈（f1.4–f2.0模拟），前景虚焦元素常态化 |
| 多重曝光 | 关键情感节点使用2–3层叠画，透明度30–50% |
| 慢动作 | 核心情感镜头 120fps 转 24fps，关键帧拉伸 |
| 手持抖动 | 轻微摇晃（8mm以内位移），增加真实感 |
| 字幕字体 | 手写细体中文 / 或等宽英文衬线体，白色，边缘微模糊 |
| 转场方式 | 硬切（节奏段）+ 溶解（情感段）+ 无转场叠化（意识流段） |
